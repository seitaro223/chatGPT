## 1. 全体評価

二つの前提検証を独立に再導出した結果、**研究判断の片側が崩れる**。

- **単管側の「x_eq独立効果」(ΔR²=0.110) は、result quantity循環でほぼ説明される**（再導出）。x_eqを測定パワー由来に置換するとΔR²=0.110→0.015、残差説明力0.147→0.030へ崩落。
- **バンドル側の「F1後残差はL/DH側に残る」は、clean F_form(v2)でも消えず、むしろ強まる**（再導出）。よって legacy F_form汚染の影ではない。ただしF_formがPMの計算に入っており、F_form↔L/DH=0.71のため機械的交絡が残る。
- 結果、**「単管で効く／バンドルで効かない＝単管とバンドルの差」という読みは時期尚早**。単管側の「効く」が循環で崩れるため、比較の片側が未確定（user自身の逆の読みが当たっている）。

## 2. 単管側の評価（x_eq循環性チェック）

### 2.1 v3の独立再導出（再導出 / 一致）
`computeXeq`（ST_BT05_v3.m:297-310）は **x_eq を qP_F1（モデル予測CHF）から**算出: `x_eq=(qP_F1·4·LD/G/1000 − Hsub)/hlg`。応答 `PM_F1 = qP_F1/qM` も qP_F1 を含む。
v10 `target_rows_T8_12` × r8 `tm_r124_F1_T8_14` を No_TableNo 結合（N=176）して独立再計算:

| 量 | 独立再導出 | v3報告 |
| --- | --- | --- |
| Hsub+P+Tsub R² | 0.81387261 | 0.81387261 |
| +x_eq(qP_F1) R² | 0.92420685 | 0.92420685 |
| ΔR² | 0.11033425 | 0.11033425 |
| 残差(Hsub+P+Tsub) vs x_eq R² | 0.14678769 | 0.14678769 |

→ v3は転記・計算とも忠実（**下層に支持される**, 再導出）。

### 2.2 反実仮想: x_eqを測定パワーqMから作る（再導出 / 決定的）

| x_eqの作り方 | ΔR²(Hsub+P+Tsub後) | 残差 vs x_eq R² |
| --- | --- | --- |
| **qP_F1（予測）= v3** | 0.11033 | 0.14679 |
| **qM（測定）= 反実仮想** | **0.01452** | **0.02959** |

x_eqを測定パワー由来にすると、追加説明力ΔR²は **約1/8**、残差説明力は **約1/5**（コード自身の閾値 `x_eq_remains≥0.05` を下回り `weak`）へ崩落。

### 2.3 機構（再導出）
| 相関 | 値 |
| --- | --- |
| corr(x_eq_qP, PM_F1) | +0.876 |
| corr(x_eq_qP, qP_F1) | −0.055（直接ではない） |
| corr(x_eq_qP, L/D) | +0.862 |
| corr(PM_F1, L/D) | +0.868 |
| corr(x_eq_qM, PM_F1) | −0.429（符号反転） |
| x_eq vs L/D（v3報告） | R²=0.742 |
| Hsub vs Tsub（v3報告） | R²=0.993 |

x_eq_qPがPM_F1と+0.876で並ぶのは、**両者が qP_F1（予測）と L/D構造を共有**するため。測定qM版は符号が反転し（−0.429）、L/Dとの整合も消える（+0.862→−0.238）。

### 2.4 所見
- 「単管でx_eqがHsub/P/Tsub後も独立に効く」: **下層に支持されない**（再導出）。ΔR²=0.110は予測値qP_F1の自己参照（result-quantity循環）と x_eq↔L/D共線性で大半が説明される。
- 逆の読み（user提示）「循環または共線性で説明できないか」→ **その通り**。両方が効いている。
- 補足: Hsub↔Tsub=0.993 のため `Hsub+P+Tsub` は実質 `Hsub+P`。「Tsub後」という増分の言い回しは見かけ上厳しく見えるが、Tsubは新情報をほぼ足していない。

## 3. バンドル側の評価（v1→v2 F_form差し替え）

### 3.1 F_formはPMの計算に入っているか（再導出）
v1(legacy)とv2(FformLinearRecalc)の tm_F1_* を独立再計算:

| case | F_form v1→v2 | q_P v1→v2 | PM_F1 v1→v2 |
| --- | --- | --- | --- |
| 108 | 0.66949→0.63609 (−0.033) | 3.236e6→3.408e6 | 1.06679→**1.12322** (+0.056) |
| 161 | 1.00000→1.00000 | 1.295e6→1.295e6 | 0.90884→0.90884 (0) |
| 164 | 1.34638→1.27988 (−0.067) | 1.067e6→1.124e6 | 0.89202→**0.93956** (+0.048) |

→ **F_formはq_P（従ってPM_F1）の計算に入っている**。PM自体が動く。161はF_form=1で不動。164（F_form修正最大−0.067）もPMが+0.048動いた。

### 3.2 L/DH残差は汚染の影か（再導出 / 決定的）

| R²（点群 N=58） | v1(legacy) | v2(clean) | Δ |
| --- | --- | --- | --- |
| PM_F1 vs L/DH | 0.3900 | **0.4768** | +0.087 |
| PM_F1 vs z_DNB/DH | 0.2844 | **0.4435** | +0.159 |
| PM_F1 vs z_DNB/L | 0.0783 | **0.2210** | +0.143 |
| PM_F1 vs F_form | 0.2398 | 0.2432 | +0.003 |
| PM_F1 vs x_eq | 0.0385 | 0.0322 | −0.006 |

→ user提示の切り分け基準「ほぼ動かないなら L/DH残差は本物」に対し、残差相関は**消えるどころか強まった**。よって**バンドルのL/DH側残差は legacy F_form汚染（-0.01・下流面積混入）の影ではない**: 下層に支持される（再導出）。BT14-B2でも current v2 = linear_v1（max_abs_delta=3.08e-8）。

### 3.3 ただし重大な交絡（再導出）
- **F_formはPMの計算に入り、かつ F_form vs L/DH = 0.714（v2）**。つまり PM_F1 vs L/DH の一部は「F_formがq_calcの因子で、F_formがL/DHを辿る」機械的交絡。これは単管のqP循環ほど直接的ではないが、**PMの構成項(F_form)で相関軸(L/DH)を説明している**点で同種の注意が要る。
- z_DNB/DH vs L/DH = 0.853、群は実質3つ。L/DH・z_DNB/DH・z_DNB/L・F_formは分離不能。
- clean化でPMが上振れ（108・164とも+）し、long/short分離は**鋭くなった**。F_form是正は系統差を減らさず、むしろ際立たせた。

### 3.4 所見
- 「F1後残差は F_form/DNB/L/DH側に残る」: **下層に支持される**（再導出、v2でも維持）。
- 「F_form/DNB/L/DH補正式を作らない」: **支持される**。交絡(0.71-0.85)＋3群＋F_formがPM内、で因果分離不能のため、補正式化は危険。
- 逆の読み「v2で消えるか」→ 消えない。ただし**PM内F_form×L/DH交絡**という別の弱点が露出。

## 4. 単管とバンドルの比較評価

- humanの読み（r30 9091-9102）: 単管=x_eq効く / バンドル=x_eq弱い / ∴単管とバンドルの差。
- §2より**単管側の「効く」が循環で崩れる**。比較の片側（単管）が未確定になる。
- よって「単管とバンドルの差」: **判断材料不足**（user逆の読みが当たる）。両前提が確定する前の段階。
- 別の読み（要検討）:
  1. 単管・バンドルとも「x_eqは独立に効かない／弱い」で**一貫**しているかもしれない（差ではなく一致）。バンドルBT05のx_eq弱さ（再導出: PM_F1 vs x_eq=0.032、Tsub後増分小）と、単管のqM版崩落は同方向。
  2. 161は z_DNB/L≈0.997（ほぼ出口DNB）で 108(0.738)・164(0.797)と機構が別の可能性。3群を同一軸に並べる比較自体が揺らぐ（構造確認: BT13C z_DNB_L 108=0.738/161=0.997/164=0.797）。

## 5. x_eqの扱い
- 単管: x_eq(qP_F1)由来のΔR²=0.110は**循環＋L/D共線性**で大半説明（再導出）。「前向き量だから補正入力に使える」としても、**予測CHFから作ったx_eqを予測CHFのPMの説明に使う**のは閉包の循環。
- バンドル: x_eqはTsubに対し追加説明力小（再導出、PM_F1 vs x_eq=0.032）。
- 扱い: x_eqは**独立補正変数の根拠なし**。測定パワー由来x_eqなら単管でも効果は小。診断量に留めるのが下層と整合。

## 6. L/D・熱履歴の扱い
- 単管: x_eq↔L/D=0.742。単管で見えた「x_eq効果」の残った部分もL/Dと不可分。L/D単独補正回避は支持される。
- バンドル: L/DH↔z_DNB/DH=0.853、F_form↔L/DH=0.71、3群。L/DH補正式回避は支持される。
- BMI/WAPD/Beckerはリポジトリに無く、L/D=履歴代理という読みは**文献側を下層照合できない**（判断材料不足）。

## 7. F_formの扱い
- linear_v1固定: **下層に支持される**（再導出, BT10で5マスタ+116行一致、current v2=linear_v1 3.08e-8）。
- legacyの -0.01手動補正（164_134in: cumInt(z=0.870)−0.010000厳密）・下流面積混入・別列転用（108_76in=F_form_old）は**linear_v1に伝播せず**（再導出）。
- 「F_formを原因変数にしない」: 支持される。ただし§3.3の通り**F_formはPMの構成項であり、L/DHと0.71交絡**。F_formを単なる無害な相関軸とみなすのは危うい。

## 8. 補正式を作らない判断の妥当性
- **下層に支持される**。単管(x_eq循環)・バンドル(3群・PM内F_form×L/DH交絡・L/DH↔z_DNB/DH=0.853)とも、補正式の係数を物理効果として読める分離が成立しない。
- 統計強度（出力照合, v3 §10 residual_by_table_LD）:
  - Table12 long−short: Hsub+P+Tsub=+0.2017 CI[0.123,0.280] → +x_eq(qP)で−0.0492 CI[−0.098,−0.0002]（**辛うじて0を外れる**）。Hsub三次単独=−0.028 CI[−0.102,0.046]（**0を跨ぐ**）。
  - N_long=9/表。**「Table12残差は機構特定済み」ではなく N=9・CIほぼ0跨ぎで未決**。しかもその「消去」は循環x_eqによるもの。
  - → 補正式化しない判断は強度面からも支持される。

## 9. 追加でやるべき計算（A/B/C）

## 11. 現時点で主張してよいこと
- F_form linear_v1は正本として下層と一致し、legacy汚染（-0.01等）を含まない（再導出）。
- バンドルのF1後系統差（108過大・161/164過小）と、PM_F1がL/DH/z_DNB側に相関する事実は、clean F_form(v2)でも維持される（再導出）。legacy汚染の影ではない。
- バンドルでx_eqはTsubに対し弱い（再導出）。F1(Tsub)維持・F(x_eq)非置換は妥当。
- 補正式を作らない判断は、交絡・小N・循環の三点から支持される。

## 12. まだ主張しない方がよいこと
- 「単管ではx_eqが独立に効く」：qP_F1循環を除くと崩れる（測定パワー版ΔR²=0.015）。
- 「単管とバンドルの差」：単管側前提が未確定のため早い。むしろ「両側でx_eq独立効果は弱い」一致の可能性。
- 「Table12 long残差はHsub+P+x_eqで解消＝機構特定」：N=9・CIほぼ0跨ぎ・循環x_eq依存で未決。
- 「F_formは無害な相関軸」：F_formはPMの構成項でL/DHと0.71交絡。
- L/D=熱履歴代理：下層に文献が無く照合不能。
