- F1後のP/Mが108で相対的に高く、161/164でやや低い理由は何か。
- それは x_eq の違いで説明できるか。
- それは z_DNB/DH の違いで説明できるか。
- それは z_DNB/L、すなわちDNBが上流寄りか出口寄りかで説明できるか。
- L/DH全体ではなく、DNB位置までの実効履歴長として見た方がよいか。
- F1(Tsub)は108/161/164に対して同じ意味で効いているか。
