## 13. 一次読み

```text
BT16では、BT13-B正本入力を読み、FformLinear_v1再計算後の状態を説明用に整理した。

PM_F1平均は、108が1を超える過大側、161/164が1未満の過小側に残る。

BT16では追加補正式を作らず、説明整理に留める。
```
