## 11. 異常点・注意点（事前予測）

new-only特有の注意候補（実行で重点確認）:
1. **39.01_10（Tsub=5.3K, x=+0.042, near-saturation）**: 最も収束が難しい可能性。Tsub小→Tf≈Ts付近で残差感度が高い。
2. **150.01_10（G=521, q_M=0.852MW）**: 低G・低flux極小点。外側二分探索の下限(1e5)付近に解が来る可能性。
3. **289.01_10（G=10565）・249.01_10（q_M=14.76MW）**: 高G・高flux極大点。上限(1.2e7)付近に解が来る可能性 → 上限張り付き注意。
4. **259.01_10（x=−0.459）**: 強サブクール・高G。Solver初期値からの収束に反復増の可能性。
5. **Flag=G 2点（281/40）**: PWR-like G超の点。挙動が他と大きく異ならないか（Flag=Gは抽出フラグであり物理発散要因ではない想定だが要確認）。
6. **x≈+0.05点（81/281）**: lowX上限。正クオリティ側でモデル前提から外れる可能性（PM解釈は本段階では不問）。
> いずれも「PMの良し悪し」ではなく「計算が壊れず有限値・Logが出るか」を見る。B1aで Tsub 33〜330K・G 2089〜3920・D 1.9〜7.8mm を問題なく通過済みのため、健全性の見込みは高い。ただしG=521やG=10565はB1aの範囲外で**初確認**。
